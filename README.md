# Personal Portfolio

A dark, sports-focused portfolio website built with Next.js, React, TypeScript, Tailwind CSS, and Framer Motion.

This site is designed to showcase my frontend work, especially projects related to sports, esports, analytics, and data-driven interfaces.

## Features

- Dark modern portfolio layout
- Project showcase for `mpl-my-stats` and `f1-dual-analytics`
- JSON-driven homepage content for easier updates
- Responsive design for desktop and mobile
- Smooth motion and glass-style UI treatment

## Content Editing

Most of the website content can be updated from:

`src/content/homepage.json`

You can change:

- profile information
- hero text
- project cards
- about section
- skills
- contact details

This makes it easy to update the portfolio without editing the page layout directly.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Available Scripts

- `npm run dev` - start local development
- `npm run build` - build for production
- `npm run start` - run the production build
- `npm run lint` - run ESLint

## Deployment

This project is set up for deployment on Vercel.
